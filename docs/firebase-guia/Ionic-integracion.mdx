**Ionic** y **Firebase** para el desarrollo de **aplicación móviles poderosas**.

## Qué es Ionic

Ionic es un framework de **Javascript** orientado al desarrollo de poderosas aplicaciones móviles de manera hibrida utilizando tecnologías web como **HTML**, **CSS** Y **JAVASCRIPT**. Permite desarrollar en paralelo una aplicación compatible con Android, iOS y navegadores Web. 

En su [**sitio oficial**](https://ionicframework.com/) se puede encontrar más información acerda del proceso de instalación y configuración. Ionic también es altamente compatible con otros frameworks de desarrollo web de Javascript como **React**, **Angular** o **Vue**, esta caracteristica lo convierte en una grandiosa opción para desarrollar aplicaciones moviles aplicando conceptos y conocimientos del desarrollo Web utilizando un stack moderno.

## Capacitor

Capacitor se encuentra desarrollada por el mismo equipo de Ionic. Esta herramienta permite que las aplicaciones web puedan acceder a funcionalidades nativas del dispositivo móvil, tales como la cámara, GPS, almacenamiento, notificaciones push entre otras. Actua como **un puente entre el código web y las API's nativas de Android e iOS**. 

En el [**sitio web**](https://capacitorjs.com/) oficial se puede encontrar mayor documentación acerca del proceso de instalación, configuración así como también la lista de plugins que posee los cuales permiten integrar las diversas funcionalidades nativas. 

## Plugins 

Capacitor cuenta con un amplio número de plugins, los cuales, se utilizan para cubrir las diversas funcionalidades nativas de los dispositovs móvil en las aplicaciones desarrolladas con Ionic. También existen plugins no oficiales los cuales son desarrollados por la comunidad de Ionic, estos permiten implementar funcionalidades nativas que Capacitor no posee pero también proporcionan caracteristica y funciones que se pueden utilizar para mejorar la experiencia de usuario.

## Capacitor y Firebase

Capacitor es esencial para la implementación de **firebase** dentro de aplicaciones moviles desarrolladas con Ionic, dado que provee acceso a funcionalidades nativas que son utilizadas por firebase como el almacenamiento y las notificaciones push. 

Gracias a su arquitectura basada en plugins, Capacitor simplifica la incorporación de los diversos servicios que ofrece Firebase. [**Capawesome**](https://github.com/capawesome-team/capacitor-firebase) es una colección de plugins que ofrecen soporte para una Integración profunda de firebase.
